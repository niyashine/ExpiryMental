ExpiryMental(Fridge Expiry Tracker)

A smart fridge management web application that helps users track food items, monitor expiry dates, and reduce food waste efficiently.

 Project Description
Fridge Expiry Tracker is a full-stack web application designed to help users manage food items stored in their refrigerator. 
The system allows users to add items and track  expiry dates

The goal of the project is to reduce food waste and improve household inventory management.
